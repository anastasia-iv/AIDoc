# AIDoc
AIDoc - это многофункциональный интеллектуальный помощник, созданный с целью не заменить врача, но предоставить возможность получить первичную консультацию для пользователя (студентов / медицинский работников / аналитиков здравоохранения) и ответы на вопросы с опорой на достоверные медицинские источники
